## SI473X and Raspberry Pi Pico wire up

The table below shows the Si4735 and RASPBERRY PI PICO connections 
    
| Si4735 pin     |  PICO Pin  |
| ---------------| ---------- |
| RESET (pin 15) |     GP16   |
| SDIO (pin 18)  |     GP0    |
| CLK (pin 17)   |     GP1    |
